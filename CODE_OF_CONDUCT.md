# 行動規範

## 基本方針

本プロジェクトでは、技術的な意見の違いにかかわらず、参加者が安全に協力できる環境を維持します。

歓迎する行動:

- 事実、再現手順、コードに基づいて議論する
- 相手の経験や背景を尊重する
- 指摘を具体的かつ実行可能な形で伝える
- 誤りを確認した場合に訂正する

認めない行動:

- 侮辱、脅迫、差別、嫌がらせ
- 個人情報の公開
- 議論を妨害する継続的な投稿
- その他、共同開発の場として不適切な行為

## 適用範囲

Issue、Pull Request、レビュー、Discussionsなど、本プロジェクトに関係する公開・非公開の場へ適用します。

## 対応

メンテナーは、違反投稿の編集・削除、参加制限、ブロックなど、状況に応じた対応を行います。行動規範上の問題を報告する場合は、公開Issueを避け、GitHubユーザー[@nematatu](https://github.com/nematatu)のプロフィールに記載された連絡手段を使用してください。
